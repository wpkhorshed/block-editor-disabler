This plugin for disable gutenberg editor for specific role

=== Description ===
The Block Editor Disabler Plugin allows you to easily disable the Gutenberg block editor on your WordPress site.
With this plugin, you can choose to disable the editor for specific pages, all pages, or even all posts, giving you complete control over your editing experience.


### Features
* Giving you complete control over your editing experience.
* Show all control on <strong> Block Editor Settings </strong> admin menu page.


== Installation ==

1. Upload <strong>Block Editor Disabler</strong> to the /wp-content/plugins/ directory.
2. Activate the plugin through the Plugins menu in WordPress.
